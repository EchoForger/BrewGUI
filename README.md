# BrewGUI

A Tkinter GUI for Homebrew (brew) on macOS.

<img src="assets/gui_light.png#gh-light-mode-only" width="50%">
<img src="assets/gui_dark.png#gh-dark-mode-only" width="50%">

## Install

```bash
pip install brewgui
````

## Run

```bash
brewgui
```

> Requires Homebrew installed and available as `brew`.
